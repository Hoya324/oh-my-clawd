Claude Pet

A Tamagotchi-style menu bar companion that lives alongside your Claude Code sessions.

12 collectible pets. 3 muscle stages. Condition-based unlocks.

Features

Your pet reacts to your coding activity in real-time:

  • More agents running = your pet gets buffer (Normal → Buff → Macho)
  • Multiple sessions = friend pets appear in the menu bar
  • Use more AI = unlock new pets (12 total, each with unique unlock conditions)
  • Rate limit warnings via macOS notifications

Install

git clone https://github.com/Hoya324/claude-hud.git
cd claude-hud
./install.sh        # HUD status line
./pet/install.sh    # Claude Pet menu bar app

README Badge

Show your pet collection in your GitHub README:

node pet/generate-badge.mjs > docs/pet-badge.svg
git add docs/pet-badge.svg && git push

Then add to your README:
