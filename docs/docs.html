Installation

Download DMG (Recommended)

Download the latest DMG from GitHub Releases, open it, and drag ClaudePet.app to your Applications folder.

Download DMG

Manual Install

git clone https://github.com/Hoya324/claude-hud.git
cd claude-hud
# Install HUD status line
./install.sh

# Install Claude Pet menu bar app
./pet/install.sh

Quick Start

  1. Download — Get the latest DMG from GitHub Releases
  2. Install — Drag ClaudePet.app to Applications
  3. Launch — Open the app and check your menu bar

Pet System

Your pet lives in the macOS menu bar and reacts to Claude Code activity across all sessions.

Pet States

State Trigger Behavior
Sleeping No active sessions Zzz animation, eyes closed
Walking Normal usage Idle walk cycle
Running 50+ tool calls Fast movement, sweat drops
Bloated Context >= 70% Puffed up, slow movement
Stressed Rate limit >= 80% Shaking, alert icon
Tired Session > 45 min Droopy eyes, yawning
Collab 2+ agents Friend pets appear

Muscle Stages

Your pet grows based on concurrent agent count.

Stage Condition Effect
Normal 0-1 agents Default appearance
Buff 2-3 agents Slightly larger, defined muscles
Macho 4+ agents Maximum size, full muscle definition

HUD Status Line

A real-time status line showing Claude Code metrics.

HUD Format

RL:45% | S:12m | CTX:62% | T:23 | A:2 | opus-4 | Pet:Cat
Segment Description Color Logic
RL Rate Limit usage percentage Green < 50%, Yellow < 80%, Red ≥ 80%
S Session duration Green < 30m, Yellow < 45m, Red ≥ 45m
CTX Context window usage Green < 50%, Yellow < 70%, Red ≥ 70%
T Tool call count Green < 30, Yellow < 50, Red ≥ 50
A Active agent count Green ≤ 1, Yellow 2-3, Red ≥ 4
Model Active model name Static (no color change)
Pet Current pet name Static (no color change)

Configuration

The HUD status line can be toggled on or off via the install.sh script. Run ./install.sh to enable or ./uninstall.sh to disable the HUD.

Pet selection is available through the app popover. Click the pet icon in the menu bar to open the popover, then choose your active pet from the collection grid.

Update

Check for updates directly from the app popover, or download the latest DMG from GitHub Releases.

Use the Check for Updates button in the popover to see if a new version is available. If an update is found, you will be directed to the GitHub Releases page to download the latest DMG.
