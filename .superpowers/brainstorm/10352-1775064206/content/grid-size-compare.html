펫 그리드 사이즈 비교

현재 16x16 vs 더 큰 사이즈 옵션 — 같은 고양이 캐릭터를 각 해상도로 표현

A. 16×16 (현재)

현재 사용 중. 디테일 제한적.
메뉴바 표시: 32×32px (2x 스케일)

픽셀 수: 256 / 스프라이트당 ~100줄

B. 24×24 (추천)

디테일 2.25배 증가. 눈, 귀, 표정 표현력 대폭 향상.
메뉴바 표시: 36×36px (1.5x 스케일)

픽셀 수: 576 / 스프라이트당 ~150줄

C. 32×32

디테일 4배 증가. 매우 정교한 표현 가능.
메뉴바 표시: 32×32px (1x 스케일)

픽셀 수: 1024 / 스프라이트당 ~250줄

참고: 메뉴바 제약

macOS 메뉴바 높이는 약 22pt (Retina 44px). 현재 32×32px 표시가 잘 맞습니다.
• 24×24 그리드: 1.5x 스케일 → 36×36px 표시 — 메뉴바에 적합
• 32×32 그리드: 1x 스케일 → 32×32px 표시 — 메뉴바에 적합하나 Retina에서 작게 보일 수 있음
• 스케일은 메뉴바/컬렉션뷰에서 다르게 적용 가능
